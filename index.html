Rechnungs-Suche

Sie können nach der Rechnungs-Nummer suchen.
Eine Suche nach Rechnungs-Nummer erforder immer die Eingabe von

  • Rechnungs-Datum
  • Filial-Nummer
  • Rechnungs-ID

Sie können nach einer Artikel-ID suchen.

Sie können nach einem Artikel-Code suchen. Diesen finden Sie z.B. auf der Verpackung, siehe Anhang B.
